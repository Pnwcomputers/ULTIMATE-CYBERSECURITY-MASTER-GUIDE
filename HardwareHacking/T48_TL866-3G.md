# 💾 T48 TL866-3G Universal IC Programmer

## 📖 Description
The T48 (TL866-3G) is a universal device programmer supporting over 31,000 ICs. It is a staple in hardware hacking for desoldering memory chips, dumping their firmware, modifying the binary, and flashing it back.

## 🔗 Reference Links
* **Product Link:** [T48 TL866-3G Programmer on Amazon](https://www.amazon.com/dp/B0BFX9FXGV)
* **Software/Drivers:** [Xgpro Software Download (XGecu)](http://www.xgecu.com/en/)

## 🚀 Step-by-Step Example: Setup, Connect, and Use
**Scenario: Dumping the BIOS/Firmware from a Winbond 25Q64 SPI flash chip.**

### 1. Setup
* **Software:** Download and install the `Xgpro` software. (This is native to Windows. Linux/Mac users will need to run it via a Windows VM or use Wine).

### 2. Connect
* **USB:** Connect the T48 to your Windows PC.
* **Connecting the Chip:** You have two options:
  * *Option A (Desolder):* Desolder the chip from the board using hot air and place it into an SOP8 adapter. Place the adapter into the T48's ZIF socket and lock the lever.
  * *Option B (In-Circuit):* Attach an SOP8 test clip over the chip while it is still on the board, and plug the clip's ribbon cable into the T48 adapter.
* Ensure Pin 1 (marked by a dot on the chip) matches the orientation diagram printed on the T48 socket.

### 3. Use
* Launch the `Xgpro` software.
* Click the **Select IC** button at the top left.
* Type the exact model number printed on top of the chip (e.g., `W25Q64JV`) into the search bar and select it.
* Click the **Read** button (a green arrow pointing to a computer chip).
* A dialog box will appear; click **Read** again. A progress bar will fill up.
* Once successful, the main window will populate with Hexadecimal data.
* Go to **File -> Save As**, and save the file as `firmware_dump.bin`. You can now analyze this file using `binwalk` in Kali Linux.
