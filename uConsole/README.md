# uConsole Setup Guide

## Rex's Kali or Trixie + HackerGadgets AIO v2 Board: CM4 & CM5 Configurations

Complete setup guides for building a field-deployable hacking and SIGINT platform using the ClockworkPi uConsole with a Raspberry Pi CM4 or CM5, Rex's community images (Kali Linux or Debian Trixie), and the HackerGadgets AIO v2 extension board.
