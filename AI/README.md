This directory contains **resources, checklists, scripts, and methodologies** focused on the intersection of artificial intelligence (AI), machine learning (ML), generative AI (GenAI) and cybersecurity.

**What You‚Äôll Find Here:**
- AI/ML use-case frameworks for threat detection, anomaly detection, and SOC automation

### Purpose

These resources are designed to:
- Provide **methodology and structure** for integrating AI/ML in cybersecurity workflows
